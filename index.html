Miért Hunjara?

A név a Hun (magyar identitás) és a Manjaro (technológiai alap) ötvözete. A Hunjara nem csupán egy újabb kinézet: ez egy technikailag modern, bloat-mentes és profi módon kezelt disztribúció, amely szakít a hagyományos megoldásokkal ott, ahol a hatékonyság azt megkívánja.

Az Elképzelés

Bloat-mentesség

Csak a legszükségesebb KDE Plasma alapcsomagok. Nincs felesleges irodai csomag vagy előre telepített szoftver-szemét.

Modern Boot

A GRUB helyett a Limine bootloaderre építünk: gyorsabb indítás, egyszerűbb és szebb grafikai felület.

Saját Ökoszisztéma

Minden módosítást saját csomagokba (.pkg.tar.zst) szervezünk, melyeket a Hunjara GitHub tárolója kezel.

Hunjara Repository

A Hunjara specifikus eszközök (branding, limine-manager, hookok) elérése meglévő Manjaro rendszeren:

# Nyisd meg a konfigurációt:
sudo nano /etc/pacman.conf

# Illeszd a végére:
[hunjara]
SigLevel = Optional TrustAll
Server = https://hunjara.github.io/hunjara-repo/x86_64
